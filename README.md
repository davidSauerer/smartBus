# smartBus

A personal project documenting the build of a camper vehicle conversion — covering the full journey from vehicle selection through electrical, insulation, water, heating, and interior build-out.

The vehicle type is not yet decided; all options are on the table (panel vans, minibuses, light trucks, and others). Decisions are documented as Architecture Decision Records (ADRs) with explicit requirements, options considered, and reasoning.

## Repository Structure

```
adr/          Architecture Decision Records — one file per major build decision
.claude/      Claude Code configuration (commands, settings)
```

## Use Cases

This build is designed around three use cases, in priority order:

1. **Monthly work commute Berlin ↔ Frankfurt** — ~550 km one way, year-round including winter, 2–5 day campsite stays
2. **European vacations with girlfriend** — spring–autumn, 2 people, campsites + occasional off-grid
3. **Mobile office across Germany** — full remote workdays, frequent video calls
